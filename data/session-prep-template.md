# Session Prep — Framework de 8 Pasos (Sly Flourish)

> **Filosofía:** Prep = ingredientes pre-cocinados para improvisar la comida en la mesa.
> No es un guión. Es un menú de posibilidades.
> Tiempo estimado: 15-30 min.

---

## 1. Revisar los Personajes

Releer nombres, trasfondos, motivaciones y vínculos de cada PJ.
Buscar formas de conectar el contenido de la sesión con algo personal de cada uno.

| PJ | Jugador | Motivación actual | Algo pendiente / gancho personal |
|----|---------|-------------------|----------------------------------|
|    |         |                   |                                  |

**Pregunta clave:** ¿Hay algún momento en esta sesión que pueda ser *personal* para un PJ específico?

---

## 2. Inicio Fuerte (Strong Start)

Una sola escena que arranca con acción, tensión o una decisión interesante.
Sin preámbulos. Sin "¿qué hacen?". Los jugadores ya están *en medio* de algo.

> **[Escribe aquí la escena de apertura — qué ven, oyen, sienten al abrir los ojos]**

**Tipos de inicio:** combate en curso, descubrimiento inquietante, mensaje urgente, confrontación social, peligro ambiental.

---

## 3. Escenas Posibles

3-5 escenas como ingredientes sueltos. No son actos de una obra — son posibilidades.
El DM elige cuáles usar y en qué orden según lo que hagan los jugadores.

Para cada escena: **qué pasa** + **qué obstáculo hay** + **qué se puede ganar**.

- **[Escena 1]** — Objetivo: [...]. Obstáculo: [...]. Ganancia: [...].
- **[Escena 2]** — Objetivo: [...]. Obstáculo: [...]. Ganancia: [...].
- **[Escena 3]** — Objetivo: [...]. Obstáculo: [...]. Ganancia: [...].

**Calibrar por duración:**
- Sesión 2h → 2-3 escenas
- Sesión 3-4h → 4-5 escenas

---

## 4. Secretos y Pistas (máx. 10)

Frases cortas que revelan algo sobre NPCs, lugares, eventos o historia.
**No atarlas a un lugar o tirada específica** — el DM las entrega cuando los jugadores hacen algo relevante, por cualquier camino.

1. [Secreto/Pista]
2. [Secreto/Pista]
3. [Secreto/Pista]
4. [Secreto/Pista]
5. [Secreto/Pista]

**Regla de oro:** Nunca un secreto con un único camino para descubrirlo. Si solo hay una llave, el secreto no existe.

---

## 5. Locaciones Fantásticas

2-3 lugares que el party podría visitar. Descripciones sensoriales (máx 2-3 sentidos por lugar).
Incluir elementos interactivos: cosas que se pueden escalar, romper, activar, investigar.

### [Nombre del lugar]
- **Aspecto:** [qué ven]
- **Sensación:** [qué sienten/huelen/oyen]
- **Lo fantástico:** [qué lo hace memorable o extraño]
- **Interactivo:** [qué pueden tocar, mover, romper]

---

## 6. NPCs Importantes

Para cada NPC activo en la sesión: qué quiere + cómo suena + un detalle distintivo.
No preparar diálogos completos — solo el arquetipo y la motivación.

| NPC | Ubicación | Qué quiere | Cómo suena | Detalle |
|-----|-----------|------------|------------|---------|
|     |           |            |            |         |

**Lista de nombres de reserva** (para NPCs improvisados):
- [Nombre 1], [Nombre 2], [Nombre 3]

---

## 7. Monstruos / Encuentros

Stat blocks relevantes. Para cada encuentro potencial, anotar:
- **Motivación del monstruo:** ¿huye? ¿se rinde? ¿pide refuerzos?
- **Terreno:** cobertura, terreno difícil, elevación, elementos ambientales
- **Bypass:** ¿se puede evitar con diplomacia, sigilo, o creatividad?

| Monstruo | CR | Motivación | Táctica | Bypass posible |
|----------|----|------------|---------|----------------|
|          |    |            |         |                |

---

## 8. Recompensas

No solo oro — también información, aliados, reputación, acceso.
Elegir items que creen nuevas capacidades, no solo bonos numéricos.
Mezclar consumibles (pociones, scrolls) con items permanentes.

- **[Recompensa 1]:** [condición para obtenerla]
- **[Recompensa 2]:** [condición para obtenerla]

---

## Extras opcionales

### Momento Pivote
La decisión, confrontación o revelación central. Lo que hace la sesión memorable.
> [¿Qué cambia en el mundo o en los personajes después de esto?]

- [Algo sin resolver que tire al party hacia adelante]

## Checklist rápido pre-sesión

- [ ] Releí los PJs y sus motivaciones
- [ ] Tengo un inicio fuerte listo
- [ ] Tengo 3-5 escenas como ingredientes
- [ ] Tengo al menos 5 secretos/pistas sueltas
- [ ] Los lugares tienen descripciones sensoriales
- [ ] Los NPCs tienen motivación + voz
- [ ] Los stat blocks están a mano
- [ ] Las recompensas están pensadas
